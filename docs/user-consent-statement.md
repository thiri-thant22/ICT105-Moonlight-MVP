# User Consent Statement

This Budget Tracker prototype is created for ICT105 learning purposes. We invite users to test the prototype and provide feedback on its usability and functionality. The information collected will be used only to evaluate and improve the project prototype. Users should not enter real financial details, passwords, or other sensitive personal information. Participation is completely voluntary, and users may stop the testing at any time.

## Data Collected
- Name/Nickname or Anonymous ID : Kelly
- User Role : University Student
- Feedback Score (1–5) : 4
- Usability Comments : The Budget Tracker is easy to use, but adding more expense categories would make it better.
- Task Completion Result : Successfully completed all assigned tasks.
- Time Taken to Complete the Task (optional) : 3 minutes (or 180 seconds)

## Data Use
The collected data will be used only for:
- Class project analysis
- Prototype evaluation and improvement
- Validation of the MVP experiment
- Final project documentation and presentation

No personal or sensitive financial information will be shared with third parties.
