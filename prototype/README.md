# Prototype Folder

This folder contains the first working prototype for the Budget Tracker project developed for Lab 10.

The prototype is a frontend-only web application built using HTML, CSS, and JavaScript. It uses the browser's Local Storage to save and manage expense records, allowing users to test the main features without requiring a backend database.

## Main Features
- Homepage
- Add Expense
- Expense Records
- Expense Details
- Dashboard
- Admin Management
- Search and Filter
- Local Storage Data Management

## Required Files
- index.html
- input-form.html
- record-list.html
- detail-view.html
- dashboard.html
- admin-view.html
- style.css
- script.js

## Prototype Notes
- Built using HTML, CSS, and JavaScript.
- Uses browser Local Storage to store sample expense data.
- No backend or online database is required.
- Sample data should be used instead of real financial information.
- The prototype is intended for learning, testing, and demonstration purposes only.

If the project is later connected to Firebase, Supabase, PHP/MySQL, or another backend service, the Local Storage implementation may be replaced while keeping the same user interface and workflow.
