Wireframe Quality Rules
-Use the same navigation bar on every screen.
-Use the same color theme (for example, blue and white).
-Use consistent fonts, icons, and buttons.
-Show realistic sample expense data (not empty boxes).
-Every wireframe should support the Budget Tracker workflow.
-The wireframes should be connected and easy to navigate.
-Create the wireframes using Figma, Canva, PowerPoint, HTML/CSS, or another suitable tool.
