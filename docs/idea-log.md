# Initial IT Venture Idea Log

## Idea 1: Budget Tracker App

### Problem Area
Many people cannot manage their expenses properly and often overspend.

### Target Users
Students and young professionals who want to control their spending.

### Current Alternative
Users record expenses in notebooks, spreadsheets, or rely on memory.

### Proposed IT Solution
A simple budget tracker application that allows users to record daily expenses, categorize spending, and monitor their budget.

### Possible Technology
Web application, mobile application, database, dashboard.

### Why This is Suitable
The project can be developed as a simple web prototype with basic expense tracking and reporting features within one semester.

---

## Idea 2: Student Study Planner

### Problem Area
Many students struggle to manage assignments, exams, and study schedules effectively.

### Target Users
University and high school students.

### Current Alternative
Students use paper planners, calendars, or reminder applications.

### Proposed IT Solution
A study planner application that helps students organize tasks, deadlines, and study schedules.

### Possible Technology
Web application, calendar integration, database, notification system.

### Why This is Suitable
The application can be prototyped with task management and scheduling features using basic web technologies.

---

## Idea 3: Campus Event Finder

### Problem Area
Students often miss campus events because information is scattered across different platforms.

### Target Users
University students and student organizations.

### Current Alternative
Students rely on social media posts, posters, or word-of-mouth communication.

### Proposed IT Solution
A centralized platform where students can browse, search, and register for campus events.

### Possible Technology
Web application, database, event management system, dashboard.

### Why This is Suitable
The project can be developed as a simple event listing and registration prototype during the semester.
