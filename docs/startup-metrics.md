# Startup / Product Metrics

## 1. Metrics Summary

| Metric ID | Metric Name | Metric Type | Why This Metric Matters | Formula / How to Calculate | Data Source | Prototype Screen |
|-----------|-------------|-------------|-------------------------|----------------------------|-------------|------------------|
| M-01 | Total Expense Records | Usage | Measures how many expense records users create | Count all expense records | Local Storage | Expense List |
| M-02 | Total Income | Financial | Shows the user's total income | Sum of all income records | Local Storage | Dashboard |
| M-03 | Total Expenses | Financial | Shows the user's total spending | Sum of all expense records | Local Storage | Dashboard |
| M-04 | Remaining Balance | Financial | Helps users monitor available money | Total Income − Total Expenses | Dashboard Calculation | Dashboard |
| M-05 | Most Used Expense Category | Usage | Identifies which spending category is used most | Count records by category | Expense Records | Dashboard |
| M-06 | Task Completion Rate | Validation | Measures whether users can successfully complete the prototype | Completed Tasks ÷ Total Tasks × 100 | Testing Results | Prototype Testing |

## 2. Metrics Interpretation

The collected metrics show that the Budget Tracker Website can successfully record and organize users' financial data. The dashboard provides useful information such as total income, total expenses, and remaining balance. Users are able to complete the main tasks with a high success rate during prototype testing. The expense category metric helps users understand their spending habits. Future improvements should focus on responsive design, stronger validation, and faster search and filtering.

## 3. Link to Final Prototype

The above metrics will be demonstrated in the final prototype through the Dashboard, Expense List, and Add Expense pages. Data is stored using Local Storage, allowing the system to calculate totals, display spending summaries, and monitor user activity in real time during the prototype demonstration.
