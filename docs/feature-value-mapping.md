## Reflection

### Which feature creates the strongest user value?

The **Dashboard Summary** creates the strongest user value because it helps users quickly understand their total spending, remaining budget, and spending categories. This directly helps students and young professionals avoid overspending and manage their money better.

### Which feature creates the strongest business or operational value?

The **Add Expense** feature creates the strongest business and operational value because it collects the main data needed by the Budget Tracker system. Without expense records, the system cannot calculate total spending, remaining budget, or display dashboard summaries.

### Which required feature is still weak or unclear?

The **Search and Filter Expenses** feature is still slightly unclear because the team needs to decide which filter options should be included in the MVP. Possible filters include category, date, amount range, and keyword. For the first prototype, the team will focus on category and date filters because they are the most useful and easiest to implement.
