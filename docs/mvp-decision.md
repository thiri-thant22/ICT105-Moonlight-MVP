# Lab 08 - MVP Decision

## 1. Decision

Choose one:

- [ ] Continue with the current MVP direction

## 2. Evidence Supporting the Decision

The usability test showed that most users could complete the main tasks successfully. Users understood the purpose of the system and found the interface easy to use. However, several users suggested improving button labels, navigation, and adding clearer status messages.

## 3. Requirements to Keep

| Requirement ID | Reason |
|---|---|
| FR-01 | Homepage is easy to understand and users know the purpose of the system. |
| FR-03 | Users can submit a report without major problems. |
| FR-05 | Search function works correctly and helps users find information. |

## 4. Requirements to Improve

| Requirement ID | Problem Found | Improvement Needed |
|---|---|---|
| FR-07 | Some users needed more information on the detail page. | Add clearer descriptions and labels. |
| FR-08 | Status information was not obvious. | Improve status display with better colors and icons. |

## 5. Prototype Changes Before Next Lab

- Improve button labels and navigation.
- Add clearer status indicators.
- Improve the item detail page.
- Fix minor UI alignment issues.
- Increase font size for important information.
